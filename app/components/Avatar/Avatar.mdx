# Avatar

O componente **Avatar** é utilizado para exibir a imagem de perfil de uma pessoa. Caso não seja fornecida uma imagem, ele exibe as iniciais do nome ou um ícone padrão de usuário. Esse componente é útil em diversos contextos, como interfaces de usuários e sistemas de login.

<Subtitle>Explore abaixo as diferentes variantes e propriedades.</Subtitle>

## Exemplo Padrão

Se um nome é fornecido e nenhuma imagem está disponível, as **iniciais** serão exibidas automaticamente:

<Canvas of={Story.Default} />

## Avatar com Imagem

Quando uma imagem é passada via `src`, ela é exibida no lugar das iniciais. Veja o exemplo abaixo:

<Canvas of={Story.WithImage} />

## Avatar Sem Nome ou Imagem

Caso nem um nome nem uma imagem sejam fornecidos, o ícone de usuário padrão será mostrado:

## Tamanhos Diferentes

O avatar pode ser exibido em três tamanhos diferentes: **pequeno (sm)**, **médio (md)** e **grande (lg)**.

<Canvas of={Story.DifferentSizes} />

## Avatar Quadrado

O componente permite configurar bordas quadradas, caso você não deseje o formato circular padrão.

## Como Usar

Aqui está um exemplo de como utilizar o componente em um projeto React:

```tsx
import { Avatar } from "./Avatar";

export default function App() {
  return (
    <div className="flex space-x-4">
      <Avatar name="João Silva" size="md" />
      <Avatar src="/avatar_image.jpeg" size="lg" />
      <Avatar />
    </div>
  );
}
