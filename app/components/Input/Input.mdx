# Componente de Entrada (Input)

O componente **Input** é um campo de entrada flexível e altamente configurável, ideal para coletar dados do usuário. Ele suporta vários **estilos** e **variantes**, tornando-o adaptável a diferentes contextos e designs.

## Como Usar

Este componente pode ser facilmente integrado ao seu projeto React. Aqui está um exemplo básico:

```tsx
import Input from "./Input";

export default function App() {
  return (
    <div className="p-4">
      <Input
        label="Nome"
        placeholder="Digite seu nome"
        variant="default"
      />
    </div>
  );
}
