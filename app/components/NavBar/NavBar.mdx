# NavBar

O componente **NavBar** é uma barra de navegação moderna e responsiva, projetada para ser flexível e altamente personalizável. Ele permite que você defina links de navegação, ajuste cores e configure o comportamento em diferentes tamanhos de tela.

## Como Usar

Este componente pode ser facilmente integrado em seu projeto com Next.js ou React. Veja um exemplo de uso:

```tsx
import NavBar from "./NavBar";

export default function App() {
  return (
    <div>
      <NavBar
        logo="MyApp"
        links={[
          { name: "Home", path: "/" },
          { name: "About", path: "/about" },
          { name: "Services", path: "/services" },
          { name: "Contact", path: "/contact" },
        ]}
        bgColor="bg-gray-800"
        textColor="text-white"
        hoverColor="hover:text-gray-300"
      />
      <main className="p-8">
        <h1 className="text-3xl font-bold">Bem-vindo ao MyApp!</h1>
        <p>Este é um exemplo básico de como usar a NavBar.</p>
      </main>
    </div>
  );
}
