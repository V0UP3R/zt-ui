# Componente de Alternância (Switch)

O componente **Switch** é um botão de alternância flexível, ideal para controlar estados binários (como ativar ou desativar algo). Ele oferece várias **variantes de tamanho e cor**, tornando-se versátil para diferentes contextos.

## Como Usar

Este componente pode ser facilmente integrado ao seu projeto React. Aqui está um exemplo básico:

```tsx
import Switch from "./Switch";

export default function App() {
  return (
    <div className="p-4">
      <Switch
        checked={false}
        size="md"
        onColor="bg-green-500"
        offColor="bg-gray-300"
        labels={{ on: "Ligado", off: "Desligado" }}
        onChange={(checked) => console.log(checked)}
      />
    </div>
  );
}
