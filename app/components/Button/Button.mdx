# Botão

Este é um componente de botão altamente flexível, utilizado para disparar ações ou navegar entre seções. Ele oferece múltiplos **estilos**, **tamanhos** e pode ser configurado para diferentes contextos de uso, como botões com ícones, animações e mais.

<Subtitle>Explore abaixo as diferentes variantes e propriedades.</Subtitle>

## Variantes Principais

O componente suporta uma variedade de estilos, como `fill`, `ghost`, `animated` e até um estilo de **carousel** para botões mais complexos. Veja o exemplo padrão abaixo:

## Como Usar

Este componente pode ser facilmente integrado ao seu projeto React. Aqui está um exemplo básico:

```tsx
import Button from "./Button";

export default function App() {
  return (
    <Button variant="fill" color="primary" size="md">
      Clique Aqui
    </Button>
  );
}
