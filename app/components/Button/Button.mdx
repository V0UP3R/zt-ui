## Props

O componente `Button` aceita as seguintes props:

- `children` (ReactNode): O conteúdo do botão, normalmente texto.
- `className` (string): Classes adicionais do Tailwind CSS para estilização personalizada.
- `variant` (string): A variante do botão que altera seu estilo. Pode ser um dos seguintes:
  - `faded`
  - `ghost`
  - `light`
  - `bordered`
  - `flat`
  - `shadow`
- Outras props que podem ser passadas diretamente para o botão (como `onClick`, `disabled`, etc.).

## Acessibilidade

Certifique-se de que os botões tenham texto descritivo ou elementos acessíveis para melhorar a usabilidade e a acessibilidade do seu aplicativo.
