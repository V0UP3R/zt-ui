# Divisor

O componente **Divider** é uma ferramenta versátil utilizada para separar visualmente seções de conteúdo. Ele é altamente configurável e pode ser personalizado com diferentes estilos, cores e margens.

## Como Usar

Este componente pode ser facilmente integrado ao seu projeto React. Aqui está um exemplo básico:

```tsx
import Divider from "./Divider";

export default function App() {
  return (
    <div className="p-4">
      <h1 className="text-2xl font-bold">Seção 1</h1>
      <p>Conteúdo da seção 1...</p>

      <Divider text="Ou" />

      <h1 className="text-2xl font-bold">Seção 2</h1>
      <p>Conteúdo da seção 2...</p>
    </div>
  );
}
